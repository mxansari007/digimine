What makes this class-centric

  • Subject vs section. Students see the subject (so CSE-A never repeats); teachers see the section (same subject across CSE-A/B). Same data, two lenses.
  • The gauge does double duty — “my standing #5/42” for students, “class avg 64” for teachers.
  • Gamification = class leaderboard — rank in cohort, you-vs-class-average bar, streaks. Not floating XP.
  • Teacher’s at-risk panel uses the real risk score (band + reason) — the warm flare flags trouble.
  • Run it on the go — Announce, Assign existing content + deadline, Add students, nudge non-starters.
  • Authoring stays on web — building tests/questions is a desktop job; mobile is monitor + communicate + assign.
